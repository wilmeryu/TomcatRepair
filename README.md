# Tomcat检测工具
具体的功能。可以检测指定Tomcat端口，1个或多个（因为一个服务器有时候跑了8081 8082 8083 8084 ...）如果8081卡死了，不判断直接杀java的话，其他跑着的项目也挂了。。。，但是项目都是内网，万一我外出或者不在身边，无法及时处理。所以就弄了这个，初衷就是为了能判断8081是否能够访问，不能访问的话就根据端口号杀进程，然后重启项目。完全放心啊！
